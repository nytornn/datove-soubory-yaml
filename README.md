# Návod

- každý film má svoje vlastní identifikační číslo **id** 
- > *id: 1*
- jméno je v **name** 
- > name: "No Time To Die"
- žánr můžete zapsat do **genre** 
- > *genre: "Thriller"*
- pro rok vydání slouží **year**
- > *year: "2021"*
- poslední v databázi je **country**, která slouží k pojmenování země, kde se film odehrává 
- > *country: "USA"*
